# abc-kids-home 🧸

ABC Kids — the home launcher; pick a game to play!

The home/launcher MFE (ADR-058): renders a tile per registered game and
contributes the `main` + `info` slots the games compose into. Selecting a
tile drives the control plane via updateControlPlaneState (ADR-057).
Generated by `scripts/generate-games.mjs` — do not edit by hand.
